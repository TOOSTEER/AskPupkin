ДЗ2 - обработка HTTP запросов

Выполненные требования
Основные страницы приложения
Главная страница (/) - список новых вопросов 

Популярные вопросы (/hot/) - список "лучших" вопросов 

Вопросы по тегу (/tag/python/) - фильтрация по тегам 

Страница вопроса (/question/1/) - вопрос с ответами 

Форма входа (/login/) - аутентификация 

Форма регистрации (/signup/) - регистрация 

Создание вопроса (/ask/) - форма нового вопроса 

Технические требования
Маршрутизация URL  - настроены именованные маршруты

Шаблонизация  - наследование шаблонов, include

Пагинация  - универсальная функция paginate

"Рыбные" данные  - данные для демонстрации

Статические файлы  - CSS, изображения

Проверка всех страниц:

Главная - http://127.0.0.1:8000/ - список 30 вопросов с пагинацией

Популярные - http://127.0.0.1:8000/hot/ - вопросы с высоким рейтингом

По тегу - http://127.0.0.1:8000/tag/python/ - вопросы с тегом "python"

Вопрос - http://127.0.0.1:8000/question/1/ - детали вопроса + ответы

Вход - http://127.0.0.1:8000/login/ - форма аутентификации

Регистрация - http://127.0.0.1:8000/signup/ - форма регистрации

Создание - http://127.0.0.1:8000/ask/ - форма нового вопроса
