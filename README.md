Технические проблемы и решения
Проблема 1: Ошибки кодировки
Симптом: UnicodeDecodeError в файлах с русским текстом
Решение: Переписали все файлы с английским текстом, устранив проблемы кодировки

Проблема 2: Отсутствие таблиц в БД
Симптом: no such table: app_tag при обращении к URL
Решение: Полный сброс БД, пересоздание миграций, гарантированное применение

Проблема 3: Команда fill_db не регистрируется
Симптом: Unknown command: 'fill_db'
Решение: Проверка структуры management/commands, создание init.py файлов

Проблема 4: Целостность данных при наполнении
Симптом: UNIQUE constraint failed при повторном запуске
Решение: Добавление очистки базы перед наполнением, обработка исключений
